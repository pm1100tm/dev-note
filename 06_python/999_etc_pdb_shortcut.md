# 🚀 Pdb 단축키

- breakpoint() 를 코드 단에 삽입

| 명령       | 단축 | 역할                                                                                                   |
| ---------- | ---- | ------------------------------------------------------------------------------------------------------ |
| `list`     | `l`  | **현재 실행 중인 소스 코드 주변을 출력**한다. 기본적으로 현재 줄 기준 앞뒤 몇 줄을 보여준다.           |
| `next`     | `n`  | **다음 줄로 이동한다 (step over)**. 함수 호출이 있어도 **함수 내부로 들어가지 않고** 한 줄로 실행한다. |
| `step`     | `s`  | **함수 내부로 진입한다 (step into)**. 현재 줄에서 함수가 호출되면 그 함수의 첫 줄로 이동한다.          |
| `return`   | `r`  | **현재 함수가 끝날 때까지 실행**하고, 호출한 쪽으로 바로 돌아간다.                                     |
| `continue` | `c`  | **다음 breakpoint 까지 계속 실행**한다. breakpoint 가 없으면 프로그램이 끝날 때까지 실행된다.          |
| `print`    | `p`  | **변수 값이나 표현식을 출력**한다. Python `print()` 와 다르게 디버거 컨텍스트에서 평가된다.            |
| `pp`       | `pp` | **pretty print**. dict, list, 객체 등을 **가독성 좋게 출력**한다.                                      |
| `where`    | `w`  | **현재 스택 트레이스 전체를 출력**한다. 어떤 함수 호출 경로로 여기까지 왔는지 한눈에 볼 수 있다.       |
| `up`       | `u`  | **상위 스택 프레임으로 이동**한다. (호출한 쪽 함수로 이동)                                             |
| `down`     | `d`  | **하위 스택 프레임으로 이동**한다. (다시 안쪽 함수로 이동)                                             |
