# 인텔리제이 커맨드라인 실행

맥북 로컬에서 특정 폴더에 들어가서 IntelliJ IDE 를 켜고자 한다.

```
IntelliJ 에서 Tool > Create Command-Line Launcher 메뉴 클릭
```

클릭하면 아래와 같음 메세지를 볼 수 있다.

```
To make the IDE accessible from the command line, please add
'/Applications/IntelliJ IDEA.app/Contents/MacOS' to the $PATH variable and
use 'idea' to run commands.
```

## Command Line Launcher 설정하기

```shell
echo 'export PATH="/Applications/IntelliJ IDEA.app/Contents/MacOS:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

그 후, Terminal 에서 원하는 프로젝트 또는 폴더로 이동 후

```shell
idea .
```

확인용:

```shell
which idea

# 대략 아래와 같이 표시되면 정상
/Applications/IntelliJ IDEA.app/Contents/MacOS/idea
```

## 위와 같이 했는데 안된다?

```shell
❯ idea .
zsh: command not found: idea

❯ which idea
idea not found
```

IntelliJ 를 공식 홈페이지에서 .DMG 파일을 받아서 설치했다면 되어야 한다.

source ~/.zshrc 를 실행한 터미널이 맞는지 확인할 필요가 있다. 세션이 다른 터미널이라면,
껐다가 하면 동작 한다.
