## 프로젝트 개요

> 📒 쇼퍼하우스 LG전차 커머스 솔루션 PoC

Smart TV 환경에서 사용자의 질의를 분석하여 벡터 기반 유사도 검색을 적용하여, 최적의 상품을 검색/제공하는
백엔드 API의 기술 타당성 검증하는 프로젝트입니다.

## 핵심 성과

### 2.1 핵심 성과

- 상품 데이터: 15,904건 정제 → 카테고리 분류 후 Vector DB 저장
- 상품 이미지: 60,203건 수집 → GCS 보관 및 Vision Warehouse 등록
- 리뷰 데이터: 792,485건 정제 → 상품 속성 추출 · 감정(긍/부정)·분석 및 요약 후 상품정보와 결합
- 검색 및 API : LLM 기반 의도 분석 및 가중 정렬, 상품 추천 API, 상품 이미지 검색 API, 유사질문 생성
- 검증 편의성: Swagger 문서 제공 , API 응답으로 결과 확인

```
질문 분석: 질의를 분석해 상품 상세정보 및 카테고리·속성·정렬 타입 등 정보 추출
질문 벡터: 질의 분석내용 임베딩 작업
상품 유사도 검색:
- 필터: 파트너 ID, 가격대, 카테고리
- 유사도 구성: 상품이름(25%) + 상품 상세(25%) + 리뷰 요약(25%) + 상품 태그(25%)
- 가중 합산: 상품 유사도 90% + 신뢰도/인기도 10%(평점·리뷰수·매출·긍정 비율)
- 추가 규칙: 옵션 일치 가중, 재고 유무에 따른 분류/노출 우선
```

### 주요 API

- 상품 검색 API: 텍스트 질의 기반 유사도 높은 상품 목록 및 유사질문 5개 반환
- 상품 이미지 검색 API: 이미지 기반 유사도 높은 상품 목록 반환 (Vision Warehouse 연계)
- 상품 리뷰 검색 API: 특정 상품의 리뷰·감정 분석 결과 조회

### 2.2 기대 효과

- 자연어/이미지 객체 기반 질의로 유사 상품 탐색 시간 단축
- 긍정 리뷰 비율/판매량 등 규칙 반영으로 상품 추천 품질 향상

- 텍스트/이미지 검색 API 통합 및 기능 테스트
