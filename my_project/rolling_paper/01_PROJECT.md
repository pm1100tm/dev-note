# Project Brief: Pastel Memories (MVP)

## Project Overview

**Pastel Memories** is a digital rolling paper service designed for simple, heartwarming sentiment sharing. It allows users to create, view, and contribute to collaborative message boards ("Papers") for friends, family, and colleagues. The design philosophy centers on warmth, nostalgia, and emotional connection, delivered through a soft pastel aesthetic and a minimal, focused user experience.

## Objectives

- Provide a low-friction "warm" space for digital keepsakes.
- Minimize user flow complexity to ensure accessibility.
- Create a visual identity that feels personal and sentimental.

## Target Audience

- Individuals looking to celebrate milestones (birthdays, farewells, anniversaries).
- Teams wanting to share appreciation in a collaborative, digital format.
- Users who value "digital sentiment" over utility-heavy communication tools.

## Key Features (MVP)

### 1. Minimal Onboarding

- **Direct Entry:** Users enter their name on a simple landing page to access the "Memory Garden."
- **Low Barrier:** No complex signup; focuses on the immediate act of viewing or writing.

### 2. Personalized Dashboard & Navigation

- **Navigation (LNB):**
  - **Top:** Dashboard, Archived Papers, New Memory.
  - **Bottom:** Profile Settings, Memory Stats (summarizing papers created and notes received).
- **Archived Papers:** A grid-based view of ongoing and completed rolling papers.
- **Memory Stats:** Quick glance at the user's impact (number of papers and total notes).

### 3. Rolling Paper Interaction

- **Message Cards:** Detailed views of sentiments including sender (From), recipient (To), and timestamp.
- **Categorization:** Tags (e.g., Travel, Memories, Family, Work) to organize the sentiment of messages.
- **Reply System:** Nested replies allow for dialogue and deeper connection within a paper.
- **Creation Flow:** A focused modal for writing new messages, featuring easy sender/recipient selection and category tagging.

## User Flow

1. **Landing:** User arrives at the name entry screen.
2. **Access:** User enters name and enters the dashboard.
3. **Exploration:** User views "Archived Papers" to see current collaborative boards.
4. **Contribution:** User selects a paper, views the message list, and uses the "Write a new message" pop-up to contribute.
5. **Interaction:** User can read replies or add their own to existing messages.

## Success Metrics

- **Sentiment Density:** High average number of notes per paper.
- **Interaction Depth:** Frequent use of the reply feature.
- **Retention:** Users returning to revisit "Archived Papers" as digital keepsakes.
