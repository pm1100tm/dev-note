# LLM

파스텔 메모리즈는 간단한 롤링페이퍼 MVP 서비스이다.
사용자 Flow 는 최소한으로 한다.
Flow 는 아래와 같다.
-> 1. 서비스 진입
-> 2. 이름 입력란 표시
-> 3. 이름 입력 후 엔터
-> 4. 화면 LNB 대시보드,프로필세팅,Archive Paper 메뉴 표시, 그리고 하단에 Memory Stats 박스가 있고, Papers 숫자, Noted Received 숫자 표시
-> 5. LNB 오른쪽 전체 화면 상단 Active
-> 4. 롤링 페이퍼 대시보드에서 페이퍼 확인(ex: Sarah's Farewell)
-> 5. [4.]의 페이퍼 클릭하여, 페이퍼 진입
-> 6. 메세지 카드 목록 확인 가능, 새로운 메세지 생성 가능

## 반드시 필요한 기능

- 메세지는 누가 누구에게 쓴 것인지 확인 가능하다.
- 메세지는 언제 작성한지 알 수 있다.
- 메세지에 대해서 답글을 달 수 있다.

사이트의 전체적인 컬러는 파스텔톤의 부드럽고 따듯한 색으로 해줬으면 좋겠다.
